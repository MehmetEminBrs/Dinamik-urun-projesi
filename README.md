#  Altın Fiyatına Endeksli Ürün Listeleme Uygulaması

##  Proje Hakkında
Bu proje, ürün fiyatlarını anlık altın değerine göre hesaplayan ve kullanıcıya listeleyen  web uygulamasıdır. **Next.js** ile geliştirilmiştir.

- Altın fiyatı harici bir API’den(MetalPriceAPI) anlık olarak çekilir  
- Filtreleme desteği:
- Minimum / maksimum fiyat
- Minimum popülerlik skoru  


##  Kullanılan Teknolojiler
- **Framework:** Next.js  
- **Dil:** TypeScript  
- **Stil:** Tailwind CSS  
- **Harici API:** MetalPriceAPI
-
